CUSTOMER LOGOS GUIDE
====================

Please add the following customer logo images to this folder:

1. lnt.png - Larsen & Toubro logo
2. saint-gobain.png - Saint-Gobain logo
3. dg-vaishnav.png - DG Vaishnav College logo
4. stella-maris.png - Stella Maris College logo

Recommended specifications:
- Format: PNG with transparent background
- Size: 300x150px (or similar aspect ratio)
- Resolution: 72-150 DPI
- File size: Under 100KB each

The website will automatically display these logos in the "Our Customers" section.
If logo files are not found, text placeholders will be shown instead.
